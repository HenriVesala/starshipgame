# Spaceship Game

A simple HTML5 action game where you control a spaceship fighting enemy ships.

## Features

- Control your spaceship with arrow keys
- Shoot enemies with the Space bar
- Enemy ships that move around and shoot at you
- Health and score system
- Smooth animations and collision detection
- Retro space-themed aesthetic

## How to Play

1. Open `index.html` in your web browser
2. Use the controls to survive:
   - **← → Arrow Keys** - Rotate your spaceship left/right
   - **↑ Arrow Up** - Move forward in the direction you're facing
   - **↓ Arrow Down** - Move backward
   - **Space** - Shoot in the direction you're facing
   - Destroy enemies to increase your score
   - Avoid enemy fire and collisions - one hit and it's game over!
   - The screen wraps around - you can exit one side and re-enter from the other!

## Files

- `index.html` - Main game HTML file
- `style.css` - Game styling and layout
- `script.js` - Game logic and controls
- `README.md` - This file

## Game Mechanics

- **Player Spaceship** (green) - Controlled by arrow keys, shoots with Space
- **Enemy Ships** (red) - Move randomly and shoot at you
- **One Hit = Game Over** - Any collision with enemy fire or enemy ship ends the game
- **Score** - Gain 10 points for each enemy destroyed
- **Objective** - Destroy as many enemies as possible before getting hit!

## Running the Game

Simply open `index.html` in any modern web browser. No server or build process required.

## Browser Compatibility

Works in all modern browsers that support:
- HTML5
- CSS3 (Flexbox, Gradients, clip-path)
- JavaScript ES6
